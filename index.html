Getting Started

  1. Install the Azure AI Add-on from the Google Workspace Marketplace
  2. Open your Google Sheet and find the "Azure AI" menu in the top menu bar
  3. Click "Configure Azure Settings" and enter your Azure API credentials
  4. Start analyzing text using the AZURE_ANALYZE_TEXT function in your cells

How to Use

Use the AZURE_ANALYZE_TEXT function in any cell with three optional parameters:

AZURE_ANALYZE_TEXT(text, systemPrompt, temperature)
  • text: The content you want to analyze
  • systemPrompt: Optional instructions for the AI (default: empty)
  • temperature: Controls response creativity (default: 0.7)
